10. Which of the following statements accurately describes the relationship between intrinsic value and market value in an efficient market?

A) In an efficient market, intrinsic value is always higher than market value.
B) In an efficient market, intrinsic value and market value are always equal.
C) In an efficient market, intrinsic value is always lower than market value.

Correct answer: B) In an efficient market, intrinsic value and market value are always equal.

Justification: In an efficient market, asset prices fully reflect all available information, including both public and private information. This means that the market value of an asset should accurately reflect its intrinsic value. If the market value is lower than the intrinsic value, investors can buy the asset at a discounted price. If the market value is higher than the intrinsic value, investors can sell the asset at a premium. However, in an efficient market, this should not be possible as all available information is already reflected in the asset's price, resulting in the market value and intrinsic value being equal. This is a key principle of market efficiency and understanding this relationship is crucial for investment practitioners.
